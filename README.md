SecondFit

SecondFit is an e-commerce platform focused on buying and selling preloved items, especially fashion and clothing. The app helps users give second-hand products a new life through a simple and user-friendly shopping experience.

Features
- Browse preloved products
- Add items to favorites / wishlist
- Shopping cart system
- Search and explore products
- Product detail pages
- User authentication
- Basic checkout flow
- Responsive and clean UI

Purpose
SecondFit aims to promote sustainable shopping by making it easier for people to buy and sell second-hand items instead of letting them go to waste.
